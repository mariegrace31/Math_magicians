## **Setup ⚙️**<a name="setup"></a>

Clone this repository:<br>
```sh
  git clone https://github.com/mariegrace31/math_magicians.git

```
Navigate to the folder
```
cd math_magicians
```
Checkout the branch
```
git checkout main
```

## ** 🏗️**<a name="install"></a>


```
npx eslint . --fix
```
